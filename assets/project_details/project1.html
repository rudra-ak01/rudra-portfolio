ATM Banking System

📝 Description

A Java-based standalone desktop ATM simulation system. It provides an end-to-end experience for users including secure login via MPIN, account management features like deposit, withdrawal, and money transfer, and detailed transaction tracking.

🔧 Technologies Used

  • Java (Swing for GUI)
  • MySQL for data storage
  • JDBC (Java Database Connectivity)

💡 Features

  • Secure MPIN-based authentication
  • Deposit, Withdraw, and Transfer functionality
  • User/Admin roles with transaction history
  • Real-time database connectivity using JDBC

📦 Project Type

This is an offline Java desktop application. Live preview is not applicable.
